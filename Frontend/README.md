# Customer Feedback Management System - Frontend

React/Vite frontend for the MERN Customer Feedback Management System.

## Setup

```bash
npm install
npm run dev
```

The app expects the backend API at `http://localhost:5000/api`. Change this in `.env` if needed.

## Views

- Login and registration
- Customer feedback submission with screenshots
- Customer feedback history and admin replies
- Admin analytics dashboard
- Admin feedback search/filter/reply/status management
- Admin CSV export with JWT authentication
- Admin user role/status management
